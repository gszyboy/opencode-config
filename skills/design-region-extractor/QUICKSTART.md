# Design Region Extractor - 快速使用指南

## 3 步提取你喜欢的设计

### Step 1：告诉我想提取什么

```
我喜欢 https://kaoshixing.com 的登录按钮
```

或

```
这个网站的蓝色导航栏好好看：https://xxx.com
```

---

## 方法 A：给我 URL + 描述（推荐）

**直接告诉我**：
- 网站 URL
- 你喜欢哪个部分（如：按钮、卡片、导航栏、Hero 区域）

**示例**：
```
我喜欢 https://github.com 的登录按钮
```
```
这个网站的卡片好好看：https://example.com
```
```
提取 https://linear.app 的侧边导航栏样式
```

---

## 方法 B：告诉我选择器（最准确）

如果你会使用浏览器审查元素：

1. 在目标网站按 `F12` 打开开发者工具
2. 点击左上角的"选择元素"图标
3. 点击你喜欢的那个按钮/卡片
4. 右键 → Copy → Copy selector
5. 告诉我：

```
https://example.com 选择器：#submit-btn
```

---

## 方法 C：上传截图

1. 截一张你喜欢的区域
2. 告诉我这是哪个网站的哪个部分

```
[上传截图]
这是某个网站的 CTA 按钮
```

## 常见问题

### Q：我不知道选择器怎么办？
A：没关系！告诉我 URL + 描述就行，我会尝试多种选择器。

### Q：我只想要颜色，不想要其他？
A：告诉我你只需要什么，我会只提取那部分。

### Q：提取的风格能直接用吗？
A：是的，我会生成 Tailwind 配置或 CSS 变量，可以直接复制到你的项目。
