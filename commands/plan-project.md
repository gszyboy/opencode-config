请分析项目需求，生成详细的开发计划。

## 输入

请提供或指定需求文档位置：
- 文档路径
- 或直接粘贴需求内容

## 输出

生成以下内容：

### 1. 功能模块清单
- 每个模块的优先级（P0/P1/P2）
- 核心功能描述
- 涉及的 API 和页面

### 2. 开发阶段划分
- 第一阶段：MVP（核心功能）
- 第二阶段：完善
- 第三阶段：优化

### 3. 具体任务清单
每个任务：
- 明确范围
- 预计工时
- 前置依赖

### 4. 关键决策点
- 技术选型确认
- 数据库设计（简化版）
- API 设计原则

## 约束

- MVP 只包含 P0 功能
- 每个任务 ≤ 4 小时
- 不超过 3 个开发阶段
- 中小型单体应用，不做微服务拆分

## 完成后

将开发计划保存到 `docs/development-plan.md`
